# Simple-Task-Manager-CLI
A simple terminal app to manage your daily tasks. It saves your tasks into a JSON file so they stay there even if you close the app. You can add new tasks, see your list, mark them as finished, or delete them.


# Simple Task Manager

This is a basic app that runs in your terminal. It helps you keep track of things you need to do.

## What it does:
- ✅ **Saves Tasks**: It creates a file called `tasks.json` to keep your data safe.
- ➕ **Add**: Create new tasks with a title.
- 📝 **View**: See a list of all your saved tasks.
- ✔️ **Complete**: Change a task status to "Completed".
- ❌ **Delete**: Remove a task you no longer need.
